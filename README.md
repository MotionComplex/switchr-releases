# switchr-releases

Public release feed for [switchr](https://github.com/MotionComplex/switchr) — a free macOS window switcher.

- **`appcast.xml`** — the [Sparkle](https://sparkle-project.org) update feed installed copies poll for new versions.
- **Releases** — each tagged release attaches the signed `switchr-X.Y.Z.dmg` installer.

Download the latest DMG from the [Releases page](https://github.com/MotionComplex/switchr-releases/releases/latest).
